## Framework File Summary (`Use a Framework for Developing Your Feedback Process.txt`)

This file describes feedback frameworks used as **conceptual background** for Phase 6.2 (Peer Feedback). The frameworks referenced are:
1. **COIN** — Context, Observation, Impact, Next Steps
2. **SBI** — Situation, Behavior, Impact
3. **STAR** — Situation, Task, Action, Result
4. **Pendleton Model** — Self-assess → Positive → Suggestion
5. **BOOST** — Balanced, Observed, Objective, Specific, Timely
6. **360-Degree Feedback** — Multi-source feedback
7. **Stop-Start-Continue** — Simple 3-part method
8. **Positive Sandwich** — Positive → Constructive → Positive *(primary model used in Phase 6.2)*

The **Positive Sandwich** is the main pedagogical model throughout Phase 6.2 remedials (especially B2 and C1 levels). This should inform critique/debate exercises at those levels.

## Feedback Framework Usage in Phase 6.2

The **Positive Sandwich** model (from the framework file) is the primary technique taught:
- **Structure:** Positive observation → Constructive feedback → Positive reinforcement
- **Used in:** All B2 and C1 remedial dialogue/debate tasks in Steps 1–5 of SP2
- **Key rule from spec:** Praise must be specific and sincere — generic positives invalidate the model

Other frameworks (COIN, SBI, STAR, Pendleton, BOOST, 360°, Stop-Start-Continue) are **reference material** for C1-level critique/debate tasks where students analyze and apply structured feedback methodologies.
