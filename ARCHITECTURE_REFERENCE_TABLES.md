## 📊 Part 6: Scoring Types Quick Reference

### Type 1: Binary Scoring (+1 per item)
**Used in**: Matching, gap-fill, multiple choice
**Example**:
```
8 items to match
User gets 6 correct
Score: 6 points (not 75%, just 6)
```

### Type 2: CEFR Scoring (A1-C1 = 1-5 points)
**Used in**: Writing assessments, speaking tests, open-ended tasks
**Example**:
```
Writing task evaluated by AI/Teacher
Quality assessed at B1 level
Score: 3 points (B1 = 3)
```

### Type 3: Completion Scoring (+1 for completing task)
**Used in**: Games, activities that need to be done
**Example**:
```
Wordshake game completed
Score: 1 point (not based on performance)
```

### Type 4: Hybrid Scoring (A1-C1 + bonus)
**Used in**: Complex tasks with multiple components
**Example**:
```
Writing (CEFR): 3 points (B1)
Grammar corrections: +1 point
Vocabulary range: +1 point
Total: 5 points
```

## 📊 Part 9: Scoring Algorithm (Pseudocode)

```
FUNCTION calculate_step_score(step_id, interactions):
    total_score = 0
    
    FOR each interaction IN interactions:
        IF interaction.type == "CEFR_ASSESSMENT":
            // Evaluated as A1-C1
            IF quality == A1: score = 1
            ELSE IF quality == A2: score = 2
            ELSE IF quality == B1: score = 3
            ELSE IF quality == B2: score = 4
            ELSE IF quality == C1: score = 5
        
        ELSE IF interaction.type == "BINARY_TASK":
            // Count correct answers
            score = count_correct_answers()
        
        ELSE IF interaction.type == "COMPLETION":
            // Simply completed or not
            score = 1 if completed else 0
        
        total_score += score
    
    RETURN route_by_threshold(total_score)

FUNCTION route_by_threshold(score):
    IF score < THRESHOLD_1: return REMEDIAL_A1
    ELSE IF score < THRESHOLD_2: return REMEDIAL_A2
    ELSE IF score < THRESHOLD_3: return REMEDIAL_B1
    ELSE IF score < THRESHOLD_4: return REMEDIAL_B2
    ELSE: return REMEDIAL_C1

FUNCTION evaluate_remedial(remedial_tasks):
    total_remedial_score = 0
    
    FOR each task IN remedial_tasks:
        IF task.type == "BINARY":
            points = count_correct_items(task)
        ELSE IF task.type == "CEFR":
            points = map_cefr_to_points(evaluate_quality(task))
        
        total_remedial_score += points
    
    IF total_remedial_score >= PASS_THRESHOLD[remedial_level]:
        return ADVANCE_TO_NEXT_STEP
    ELSE:
        return RETRY_REMEDIAL
```

## 🎯 Quick Decision Tree

```
User completes main step assessment
  ↓
Calculate total score
  ↓
Score within ranges? NO → Debug/Error
                    YES
  ↓
Which threshold? (A1/A2/B1/B2/C1)
  ↓
Route to corresponding remedial level
  ↓
User completes remedial tasks
  ↓
Calculate remedial score
  ↓
Score ≥ pass_threshold? YES → Advance to next step
                        NO → Show retry option
  ↓
Any more steps in phase? YES → Go to next step
                         NO → Move to next phase
```
