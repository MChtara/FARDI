## 🎓 Scoring Mechanics

### Binary Tasks (Matching, MCQ, Gap-Fill)
```
Each correct answer = +1 point
Example: 7 out of 10 correct = 7 points
```

### CEFR Assessments (Writing, Speaking)
```
A1 Level = 1 point
A2 Level = 2 points
B1 Level = 3 points
B2 Level = 4 points
C1 Level = 5 points
```

## 💡 Key Rules to Remember

1. **Scoring is additive**: Each correct item = +1 point (or CEFR level points)
2. **Remedial triggered by score threshold**: Not by performance quality
3. **Unlimited retries**: Users can keep retrying until they pass
4. **Pass once and advance**: Once you pass a remedial, you move to next step
5. **Each phase has independent steps**: Failing step 1 doesn't affect step 3 entry
6. **CEFR levels are consistent**: A1=1, A2=2, B1=3, B2=4, C1=5 across all phases

## ⚠️ Common Scenarios

### Scenario 1: Consistent Low Scores
**Steps**: Main → A1 Remedial → Pass → Step 2 → B1 Score → A2 Remedial → Pass...
**Pattern**: Gradually moving through levels

### Scenario 2: Quick Progression
**Steps**: Main → C1 Remedial → Pass → Next → C1 Remedial → Pass...
**Pattern**: Advanced users skip lower levels

### Scenario 3: Stuck in Loop
**Steps**: Main → A2 → Fail → Retry A2 → Score 17/18 → Fail → Retry...
**Pattern**: User needs practice in specific area

### Scenario 4: Jump Up
**Steps**: Main → A1 → Pass → Next → A2 Score → Next → B1 Score...
**Pattern**: User improving significantly

## 🚀 Getting Started

1. **Register & Login** → Phase 1
2. **Complete Assessment** → Automatic to Phase 2
3. **Take Step 1** → Scored
4. **Check Routing Decision** → Go to appropriate remedial or next step
5. **Complete Remedial** → Get pass/fail result
6. **If Pass**: Continue → If Fail: Retry same remedial
7. **Repeat** until Phase 6_2 completion

**Total Expected Duration**: ~4-8 weeks (varies by learner)
